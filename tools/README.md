# 8-bit Computer Visualizer

Run:

```bash
python tools/viz_8bit.py
```

Fitur:
- PC, IR, MAR, A, B, CCR
- output ports 16 buah
- Step / Run / Stop / Reset
- visualisasi program ROM test

Catatan:
- ini simulator visual Python, bukan sambung langsung ke GHDL
- cocok buat bantu bayangin kerja komputer 8-bit
